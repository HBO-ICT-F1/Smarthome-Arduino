# Smarthome Arduino

### Requirements
- (Arduino uno + Ethernet shield 2 + 9v battery) or (Wemos ESP8266)
- breadboard
- buzzer
- DHT11 (temperature and humidity)
- Resistor (10k ohm)
- 6 cables
